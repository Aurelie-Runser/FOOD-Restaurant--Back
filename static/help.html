Liste des URLs possibles

Liste des URLs pour afficher des données

  • Pour afficher les recettes : http://localhost:3000/recettes
  • Pour afficher les cuisines : http://localhost:3000/cuisines
  • Pour afficher les alergies : http://localhost:3000/alergies
  • Pour afficher les objectifs : http://localhost:3000/goals
  • Pour afficher les ingredients : http://localhost:3000/ingredients

  • Pour afficher les recettes d'une cuisine précises : http://localhost:3000/recettes/cuisine_nom/:cuisine_name
  • Exemple : http://localhost:3000/recettes/cuisine_nom/Italienne

  • Pour afficher les recettes sans allergènes : http://localhost:3000/recettes/sans_allergene

  • Pour afficher les recettes ayant un objectif précis : http://localhost:3000/recettes/goal_nom/:goal_name
  • Exemple : http://localhost:3000/recettes/goal_nom/Faible en calories

  • Pour afficher toutes les informations d'une recette précise : http://localhost:3000/recettes/all_infos/recette_titre/:recette_name
  • Exemple : http://localhost:3000/recettes/all_infos/recette_titre/Salade César

Liste des URLs pour ajouter des données

  • Pour ajouter/créer une cuisine : http://localhost:3000/ajouter_cuisine/cuisine_nom/:name
  • Exemple : http://localhost:3000/ajouter_cuisine/cuisine_nom/Irlandaise

  • Pour ajouter/créer un ingrédient : http://localhost:3000/ajouter_ingredient/ingredient_nom/:new_name/quantite/:new_quantity/unite/:new_unit
  • Exemple : http://localhost:3000/ajouter_ingredient/ingredient_nom/Oeufs/quantite/2/unite/oeuf(s)

  • Pour ajouter un ingrédient (existant) à une recette : http://localhost:3000/ajouter_ingredient_a_recette/recette_titre/:title_recette/ingredient_nom/:name_ingredient
  • Exemple : http://localhost:3000/ajouter_ingredient_a_recette/recette_titre/Salade César/ingredient_nom/Huile d'olive

Liste des URLs pour mettre à jour les données

  • Pour modifier le nom d'une recette : http://localhost:3000/modif_recette/title/recette_titre/:old_title/recette_nouveau_titre/:new_title
  • Exemple : http://localhost:3000/modif_recette/title/recette_titre/Salade César/recette_nouveau_titre/Salade Italienne

  • Pour modifier la cuisine d'une recette (à partir d'une cuisine existante) : http://localhost:3000/modif_recette/cuisine/recette_titre/:title/recette_nouvel_cuisine/:new_cuisine
  • Exemple : http://localhost:3000/modif_recette/cuisine/recette_titre/Salade César/recette_nouvel_cuisine/International

  • Pour modifier l'alérgie d'une recette (à partie d'une alérgie existante) : http://localhost:3000/modif_recette/allergy/recette_titre/:title/recette_nouvel_alergie/:new_allergy
  • Exemple : http://localhost:3000/modif_recette/allergy/recette_titre/Salade César/recette_nouvel_alergie/Gluten

  • Pour modifier une étape d'une recette (à partie d'une étape existante) : http://localhost:3000/modif_recette/etape/recette_titre/:title/etape_num/:instruction_num/etape_description/:new_etape
  • Exemple : http://localhost:3000/modif_recette/etape/recette_titre/Salade César/etape_num/3/etape_description/Servez avec des croûtons et du parmesan râpé. Régalez-vous !

Liste des URLs pour supprimer des données

  • Pour supprimer un ingrédient d'une recette : supp_ingredient_a_recette/recette_titre/:title_recette/ingredient_nom/:name_ingredient
  • Exemple : http://localhost:3000/supp_ingredient_a_recette/recette_titre/Salade César/ingredient_nom/Tomates

  • Pour supprimer une cuisine ET modifier les recettes qui avaient cette cuisine : http://localhost:3000/supp_cuisine/cuisine_nom/:name
  • Exemple : http://localhost:3000/supp_cuisine/cuisine_nom/Irlandaise
